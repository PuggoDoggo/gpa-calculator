Title
GPACalc for MATLAB

Summary (short)
Free educational MATLAB toolkit to calculate CGPA/GPA and convert between GPA and percentage for 4.0, 5.0, and 10.0 scales.

Overview (long)
GPACalc for MATLAB is a lightweight educational toolkit for GPA/CGPA planning. It helps students and educators project cumulative CGPA from current CGPA, completed credits, and semester module grades.

It supports common grading systems including US 4.0, Singapore 5.0 and 4.0 variants, Malaysia 4.0, and India 10.0 CGPA.

Included functions
- Projected cumulative CGPA calculation
- Grade label to grade-point conversion
- GPA/CGPA to percentage conversion
- Percentage to GPA conversion
- Scale metadata and aliases
